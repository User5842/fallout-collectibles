# Fallout Collectibles

I love the Fallout universe and visit the games time and time again. Throughout my playthroughs, I often keep track of collectibles I have on a piece of paper or some random note on my phone. I wanted to create something better.

Fallout Collectibles allows you to keep track of all the collectibles across Fallout 3, Fallout: New Vegas, and Fallout 4.

I hope you enjoy the website. Happy collecting!
